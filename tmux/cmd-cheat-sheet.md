# Essential Cmds
- Split Vertical: `Bind + %`
- Split Horizontal: `Bind + "`
- Change Pane Focus: `Bind + Arrow Key in Desired Dir`
- Resize Pane: `Bind + Option(hold) + Arrow in Resize Dir`
- Zoom/Unzoom Pane: `Bind + z`
- Paging Mode: `Bind + [`
- Close Pane: `Bind + x`
- Detach: `Bind + d`
- Reattach: `tmux a`
